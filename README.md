# Vipin Rai P - Portfolio Website

This is a personal portfolio and resume website built with React, Vite, and Framer Motion. It showcases my professional experience, skills, projects, and achievements as a Senior Software Engineer.

## Features
- **Responsive Design:** Optimized for all devices.
- **Interactive UI:** Smooth animations using Framer Motion.
- **Dark Mode:** Sleek, developer-focused aesthetic.
- **Dynamic Content:** Easy to update data structure.

## Tech Stack
- React
- Vite
- Framer Motion
- Vanilla CSS
